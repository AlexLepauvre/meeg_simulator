meeg_simulator.simulator module#

meeg_simulator.simulator.simulate_data(X: ndarray, noise_std: float, n_modes: int, n_subjects: int, tmin: float, tmax: float, sfreq: float, t_win: ndarray, effects: ndarray, effects_amp: ndarray | None = None, spat_cov: ndarray | None = None, ch_type: str | None = 'eeg') → List[ndarray][source]#

Simulate epoched MEG/EEG data of the specified dimensions with multivariate patterns based on the specified experimental design. Effects can be specified to be present at specific time windows, constituting “ground truth” effects for testing sensitivity and specificity of analysis methods.

Parameters:

  • X (np.ndarray) – Between-trial design matrix specifying experimental manipulations. Shape = [n_trials, n_experimental_conditions].

  • noise_std (float) – Standard deviation of additive Gaussian noise (before spatial covariance).

  • n_modes (int) – Number of modes (e.g., sensors/channels or components) in the final simulated data.

  • n_subjects (int) – Number of subjects to simulate.

  • tmin (float) – Start time (in seconds) of each epoch.

  • tmax (float) – End time (in seconds) of each epoch.

  • sfreq (float) – Sampling frequency (Hz).

  • t_win (np.ndarray) – An array of shape [n_effects, 2], specifying the (start_time, end_time) of the time window where each effect in effects should be non-zero.

  • effects (np.ndarray) – Indices of the conditions (columns in X) that actually manifest as time-locked effects. Each entry in effects corresponds to one row in t_win.

  • effects_amp (np.ndarray, optional) – Amplitude of the effects. The multivariate components are generated by sampling the beta parameter from a normal distribution across channels for a given experimental condition/contrast. This parameter specifies the variance of the normal distribution, which scales up or down multivariate patterns strength

  • spat_cov (np.ndarray, optional) – A spatial covariance matrix of shape [n_modes, n_modes]. If None, an identity matrix is used (i.e., no correlation across modes).

  • ch_type (string, optional,) – Type of the channels being simulated. Default eeg.

Returns:

A list (length = n_subjects). Each element is a mne epochs object

Return type:

List[mne.Epochs]

Raises:

ValueError – If the number of rows in t_win does not match the length of effects.
